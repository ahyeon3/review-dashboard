01왜 팀 합계가 줄었나조직 재편 타임라인

리뷰 물량이 여러 경로로 분산 처리

리뷰팀 합계 감소는 물량의 여러 부서 분산 + 인력 변동(퇴사·이관)의 결과. 현재 리뷰팀 처리분만 집계되어 합계가 줄어 보이나, 실가동 정현이(1인) 처리량은 증가.

📌 현재 처리 경로 (4갈래) — ① 리뷰팀 ② 영지부의 리뷰팀 파견 ③ 영지부 직접진행 ④ 영업부 직접진행. 신규 캠페인은 리뷰팀 ↔ 영지부 약 50:50 배분.
  • 05-14차애란·조윤정 신규 건 배정 중단 → 정현이 집중 체계 시작
  • 05-15차애란·조윤정 물량 → 리뷰팀 + 영업지원부 이관
  • 05-18전혜은 퇴사
  • 05-19전혜은 물량 → 리뷰팀·영지부 정확히 50:50 이관
  • 현재실가동 정현이 1인 · 이선주(신규) 온보딩 · 신규건 리뷰팀↔영지부 50:50 배분
