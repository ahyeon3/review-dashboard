# 리뷰팀 주간 운영 보고

**라이브 페이지:** GitHub Pages 활성화 후 자동 갱신.

## 구성

- 의사결정 필요사항 패널 (D1~D6)
- 리스크 레지스터 (severity × impact × owner)
- 처리량 현황·추이 (주차·월별·일별)
- 업무 성과·제도
- 인력 이슈 (정현이 잔류 시나리오 3종 + 퇴사 사유)
- 운영 이슈·계획 (페이백·쿠팡파트너스·throughput)
- 재무 economics (TOP5 단가·인센티브 비용)
- 액션 트래커 (담당·기한·우선순위·상태)
- 인쇄/PDF 모드 지원
